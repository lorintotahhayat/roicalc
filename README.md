# ROI Calculator

A beautiful, interactive ROI calculator built with Next.js, React, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Interactive ROI calculations
- Multi-project impact analysis
- Beautiful gradient UI with glassmorphism effects
- Responsive design
- Real-time calculations using React hooks

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide React (icons)
